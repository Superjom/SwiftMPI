A Tiny Distributed Parameter Server based on MPI
===================================================

Apps
------
* logistic: logistic regression
* word2vec: Word2Vec
* sent2vec: paragraph vector

Dependency
------------
* C++11 (gcc4.8+)
* ZeroMQ-4.0.5
* glog-0.3.3
* gtest-1.7.0
* zlib-1.2.8
* sparsehash-2.0.2
* openmpi-1.8.5
* gflags
